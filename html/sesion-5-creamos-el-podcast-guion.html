Sesión 5: Creamos el pódcast (guión)

Pasos para la elaboración de nuestro pódcast

PASO 1: Revisar guión

Debéis comprobar: 

  • Presentación.
  • Información.
  • Valores.
  • Distribuir las partes sobre las que va a hablar cada uno.

PASO 2: Ensayar

  • Ensayar el discurso y lo que va a decir cada uno (usando el guión como apoyo).

PASO 3: Grabar el pódcast

  • Para ello utilizaremos la aplicación de Edvoice.
  • Yo (maestra) os ayudaré con esta parte.

PASO 4: Revisar

Revisamos en conjunto vuestra grabación:

  • ¿Se entiende bien?
  • ¿Habéis hablado claro?
